SecureIt Configuration 1.0

CSRF extension to HackTab security tests

All web requests to the target domain will be recorded in the local extension instance and used to craft security tests in the SecureIt. Currently the url path is ignored and only the host is matched. Target domain is a REGULAR EXPRESSION.

Depending on which application parameters you test, SecureIt will run the applcation for the urls you test. If those urls store or remove data, SecureIt will create and or remove test data on the server. If in doubt, test in a sandbox.
